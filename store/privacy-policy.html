AI HOT Notifier 隐私政策

最后更新：2026年6月5日

概述

AI HOT Notifier 是一款浏览器扩展，用于监控 aihot.virxact.com 的公开资讯并推送桌面通知。我们重视您的隐私，本扩展的设计原则是最小化数据使用。

数据收集

我们不收集任何用户个人数据。

本扩展：

  • 不收集、存储或传输任何个人身份信息
  • 不使用 Cookie 或追踪技术
  • 不包含任何分析或广告 SDK
  • 不与任何第三方共享数据

本地存储

扩展使用浏览器本地存储（chrome.storage.local）保存以下数据，这些数据仅存在于您的设备上：

  • 用户偏好设置（主题、字体、轮询间隔等）
  • 资讯历史记录（标题、链接、来源、时间）
  • 已读状态标记

这些数据完全保存在本地，不会上传到任何服务器。

网络请求

扩展仅向 aihot.virxact.com 的公开 API 发起 GET 请求以获取最新资讯，不携带任何用户标识信息。

权限说明

  • alarms：用于定时轮询
  • notifications：用于推送桌面通知
  • storage：用于本地保存设置和历史记录
  • host_permissions (aihot.virxact.com)：用于访问资讯 API

联系方式

如有隐私相关问题，请通过 Chrome Web Store 的支持渠道联系我们。
