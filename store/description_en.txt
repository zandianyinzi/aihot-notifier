Detailed description:

AI HOT Notifier — Your AI News Radar

Automatically monitors the latest AI industry updates from aihot.virxact.com and delivers instant desktop notifications, keeping you informed about important developments in the AI field.

✦ Core Features

• Automatic polling for AI news with instant desktop notifications
• Click notifications or list items to jump directly to the source
• News list with title + summary for quick scanning
• Read/unread state persists across browser restarts
• Category tags (Model / Product / Industry / Paper / Tips) for quick filtering

✦ Personalization

• 3 carefully crafted themes: Obsidian Night / Dark Forest / Chrome Ink
• 3 font styles: System / Serif / Kai
• 5 font size levels: Small / Default / Large / Extra Large / Huge
• Position mode: Free Browse / Unread First
• Configurable polling interval: 2 / 5 / 15 / 30 / 60 minutes
• Configurable display range: 1 / 2 / 3 / 5 days (default: 2 days)

✦ Technical Highlights

• Manifest V3, modern architecture
• Lightweight and efficient, minimal resource usage
• Local storage only — no user data collected
• Auto-resumes polling after browser restart

✦ Data Source

News curated from aihot.virxact.com, covering AI model releases, product updates, industry news, academic papers, and usage tips — daily curated picks.
